# Roadmap Plan

## Intro

- What we will go through
- Why Prompting & Planning before Codebase Readiness

## Module 1 — Prompting & Planning

_How you communicate intent to agents_

- Basic prompting techniques
- Small, focused tasks
- Plan-then-execute workflow
- Implementation plans / strategy docs
- Specification documents

## Module 2 — Codebase Readiness

_Preconditions before agents can work effectively_

- Strong typing
- Consistent Patterns
- Naming conventions and file sizes
- Modular structure and dependencies
- Strong and simple platform

## Module 3 — Configuration

_Persistent context and agent setup_

- Context management (CLAUDE.md / AGENTS.md)
- Workflows and skills
- Code review agent

## Module 4 — Feedback Loop

_Automated signals that keep quality in check_

- Code style linter
- Advanced linters (complexity)
- Hooks and pre-commit automation
- Integration / API testing
- E2E testing
- Specialized review agents (security, performance, accessibility)

## Module 5 — Orchestration

_Scaling to multi-agent workflows_

- Sub-agents orchestration
- Delegating research and feature slices to specialized agents
